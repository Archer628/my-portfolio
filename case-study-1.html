Project Overview

The Michaels Buyer Tools mobile application is a comprehensive solution designed to streamline the purchasing process for Michaels stores nationwide. This innovative app empowers store managers and buyers with real-time inventory management, vendor communication, and purchase order tracking capabilities, all from a mobile device.

As the lead UI/UX designer, I was responsible for the entire design process from initial user research and wireframing to high-fidelity prototypes and developer handoff. I conducted extensive user interviews with store managers to understand their pain points in the procurement process, which informed the app's intuitive navigation and workflow design.

The application features a clean, modern interface that prioritizes essential buying tasks while maintaining brand consistency with Michaels' visual identity. Key challenges included designing for complex data visualization in a mobile format and creating an efficient workflow for purchase order approval processes.

Since its launch in March 2024, the Michaels Buyer Tools app has been adopted by over 500 store locations, resulting in a 40% reduction in procurement processing time and a 25% decrease in inventory discrepancies.

Design Process & Screens

User Research & Analysis

Comprehensive user research was conducted with 15 store managers across different regions to understand their procurement workflows. We identified key pain points including inefficient vendor communication, delayed approval processes, and difficulty tracking inventory levels in real-time.

Wireframing & Information Architecture

Low-fidelity wireframes were created to establish the app's core structure and navigation. The information architecture was designed to prioritize the most frequently used features like inventory checking and purchase order creation.

Visual Design System

A comprehensive design system was created with Michaels' brand colors as the foundation. The system includes typography scales, a consistent icon set, button states, and responsive layout grids.

Dashboard & Analytics

The dashboard provides store managers with an at-a-glance view of critical metrics including inventory levels, pending orders, and recent vendor communications. Data visualization was carefully designed for mobile readability.

Inventory Management

The inventory management screen allows users to check stock levels, set reorder points, and view product details with a single tap. A barcode scanner integration enables quick item lookup.

Purchase Order Creation

The purchase order creation workflow was streamlined to minimize steps while maintaining accuracy. Auto-suggest features for vendor and item selection make the process efficient.
